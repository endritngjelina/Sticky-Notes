How to Use Adobe Sticky Notes

• Click "Add Note" to create a new sticky note
• Drag notes to move them around
• Double-click to edit note content
• Right-click for context menu with advanced options
• Use color picker to change note colors
• Set priorities with the priority button
• Add tags by typing #tag in note content
• Set reminders with the reminder button
• Connect notes by clicking "Connect Notes" then clicking two notes
• Use filters to organize and find notes
• Switch between Freeform, Kanban, and Timeline views
• Export/Import to save your work
• Undo last action with Ctrl+Z
