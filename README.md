# 🗒️ Sticky Notes

A lightweight, browser-based sticky notes application built with pure 
HTML, CSS, and JavaScript. Create and manage colorful sticky notes 
directly in your browser — no frameworks, no setup required.

## 🚀 Features

- Create, edit, and delete sticky notes instantly
- Colorful and visually appealing note design
- No backend or database required — runs entirely in the browser
- Iterative versions with progressively improved UI and functionality
- Lightweight with zero external dependencies

## 📁 Project Structure

| File | Description |
|---|---|
| `sticky notes 0.html` | Base version — core note creation logic |
| `sticky notes 1.html` | Version 1 — initial styling improvements |
| `sticky notes 2.html` | Version 2 — added note editing features |
| `sticky notes 3.html` | Version 3 — enhanced UI and interactivity |
| `sticky notes 4.html` | Version 4 — additional color and layout options |
| `sticky notes 5.html` | Version 5 — improved note management |
| `sticky notes 6.html` | Version 6 — further refinements |
| `sticky notes 7.html` | Version 7 — latest and most complete version |

## 🛠️ How to Use

1. Clone or download the repository
   ```bash
   git clone https://github.com/endritngjelina/Sticky-Notes.git
   ```
2. Open any `.html` file in your browser — start with `sticky notes 7.html`
   for the most up-to-date version
3. Click to add a new sticky note
4. Type your content and customize as needed!
